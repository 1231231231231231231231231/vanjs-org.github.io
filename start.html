VanJS: Getting Started

To get started, add the line below to your script:

import van from "https://cdn.jsdelivr.net/gh/vanjs-org/van/public/van-1.0.0.min.js"

To code without ES6 modules, add the following line to your HTML file instead:

<script type="text/javascript" src="https://cdn.jsdelivr.net/gh/vanjs-org/van/public/van-1.0.0.nomodule.min.js"></script>

Alternative, you can download the files (van-1.0.0.min.js, van-1.0.0.nomodule.min.js) and serve them locally.
